# 🌍 Azure Smart Translator Web Application

> A Full-Stack AI Powered Translator built using Microsoft Azure Cognitive Services
---
## 🚀 Project Overview
Azure Smart Translator is a modern AI-powered web application that integrates:
- 🎤 Speech-to-Text
- 🌍 Multi-language Translation
- 🔊 Text-to-Speech
- ✨ Animated Modern UI/UX

Built using Python Flask for backend and Azure Cognitive Services for AI capabilities.
---

## 🧠 Features
✔ Translate text into 5+ languages  
✔ Real-time Speech-to-Text input  
✔ Azure-powered Text-to-Speech output  
✔ Animated status indicators  
✔ Modern glassmorphism UI  
✔ Loader & microphone glow effects  

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python (Flask) |
| Cloud AI | Azure Translator API |
| Speech AI | Azure Speech Services |
| Speech Input | Web Speech API |

---

## 🏗 System Architecture

User Input → Flask Backend → Azure Cognitive Services → Response → UI Display

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository
git clone https://github.com/yourusername/azure-smart-translator.git
cd azure-smart-translator

### 2️⃣ Install dependencies
pip install -r requirements.txt

### 3️⃣ Create a `.env` file
TRANSLATOR_KEY=your_key
TRANSLATOR_REGION=your_region
SPEECH_KEY=your_key
SPEECH_REGION=your_region

### 4️⃣ Run the app
python app.py
